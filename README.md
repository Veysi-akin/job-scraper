# README

### Job scraper

Job searching service using Ruby and Ruby On Rails.

### Prerequisites
Ruby 2.6.6

Rails 6.0.3.3

pg

### Installation

Follow below steps:

git clone https://github.com/vysi/job-scraper.git

cd job-scraper

bundle install

rails db:create

rails db:migrate

rails s

open http://localhost:3000

Run locally: rake scrape
